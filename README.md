# signaux3
Projet : détection de chute. Dans le cadre du cours de signaux3 2025-2026.


#Fonctionnement

- le projet se base sur de la detection de mouvement et de non-mouvement pour détecter une chute.

#Utilisation

Pour utiliser le projet, vous pouvez soit l'utilisez en temp réel, en placant la caméra de façon à voir le lit et le sol du coté ou la personne pourrait tombée
